Projects

1.National Design Innovation Network

My role in this project is to develop a virtual assistant for a restricted knowledge base for STEM VR experiments

2.Automated Cryptographic Algorithm Validation System

My main role in this project is to write code for validation cryptographic algorithms(DSA, EDDSA, RSA, KAS etc.) automatically. I am also involved in developing backend and frontend as well for automated CAVS(Cryptographic Algorithm Validation System).

3.Autonomous Navigation based on place navigation

In this project, we are trying to develop an autonomous navigation drone which can recognize path inside the building and able to map the whole building from inside. This project is still going on.

4.Implementation of Double Q-Learning algorithm on Mountain Car Game-Reinforcement Learning

n this project, I implemented the Double Q-Learning algorithm on Mountain Car Game as reinforcement learning project. I also tried to improve the algorithm learning rate.

5.Probabilistic Exponential Distribution

In this project, I developed a physical model to demonstrate the exponential distribution. It gives a real picture of the exponential distribution.

6.MNIST hand written digit recognition using CNN, RNN, and LSTM

In this project, I trained and tested CNN, RNN, and LSTM algorithm for handwritten digit recognition on the MNIST data set. I also compare the results for the given algorithms.

7.Implementation of NN, Knn, K-mean algorithm on Iris Data Set (UCI- Machine learning repository)

In this project, I trained and tested- NN, Knn, and K-mean algorithms on Iris Data Set. I also compare the results for the given algorithm.

8.Reliable Energy Balance Model for Wireless Sensor Networks

This was my major undergrad project, I extended the minor project and proposed a new model which is more efficient and reliable than the D.K. Lobiyal’s paper and implemented it.

9.Dynamic Clustering Head Node Selection Using Fuzzy C-Mean algorithm

This was my minor undergrad project in which I implemented the research paper authored by D.K. Lobiyal using NetSim Simulator.

10.Application of machine learning in prediction of types of tumor (Malignant/Benign)

In this project, I trained a machine learning model using linear regression to predict typed of the tumor (Malignant/Benign)

11.PCM with RUN

In this project, we developed a PC game. The motivation for developing this game was- teaching physics, chemistry, and mathematics to students while playing this game.
